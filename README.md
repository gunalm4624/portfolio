# Gunal M Portfolio

Portfolio website for **gunalm.design**

## Project Structure

```
src/
├── app/
│   ├── about/
│   │   └── page.tsx       # About page
│   ├── work/
│   │   └── page.tsx       # Work/Portfolio page
│   ├── ai/
│   │   └── page.tsx       # AI projects page
│   ├── contact/
│   │   └── page.tsx       # Contact page
│   ├── layout.tsx         # Root layout
│   ├── page.tsx           # Home page
│   └── globals.css        # Global styles
```

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

- **Home** (`/`) - Landing page
- **About** (`/about`) - About section
- **Work** (`/work`) - Portfolio and projects
- **AI** (`/ai`) - AI projects and experiments
- **Contact** (`/contact`) - Contact information

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- ESLint

## Domain

This portfolio is designed for **gunalm.design**
